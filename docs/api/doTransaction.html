transaction

doTransaction(request)

Dispatch a transaction command to the connected terminal. Returns immediately with an eventId; final result delivered via callbacks.

Overview

Stabilitystable
Since0.1.0
ReturnsString — eventId (UUIDv4). Use with checkStatus or cancelTransaction.

Parameters

NameTypeRequiredDescription
requestTransactionRequest✅ YesTransaction details including amount, type, merchantId, terminalId

Example request: 
{ "amount": 10000, "type": "SALE", "merchantId": "M12345", "terminalId": "T67890" }

Callbacks

CallbackTriggerPayload
onResultTerminal returned a final resultTransactionResult
onErrorTransaction failed before or during processingSdkError
onTransportEventTransport-level events (connect/disconnect/fallback)TransportEvent

Errors

VariantWhenRecoverable
NotInitializedinit() was not called❌ No
TransactionInProgressAnother transaction is already in flight✅ Yes
InvalidInputamount ≤ 0 or required field missing❌ No
TransportUnavailableNo configured transport could connect✅ Yes

Code Examples

Kotlin
val eventId = sdk.doTransaction(TransactionRequest(amount = 10_000, type = TransactionType.SALE, merchantId = "M12345", terminalId = "T67890"))
Python
event_id = sdk.do_transaction(TransactionRequest(amount=10000, type=TransactionType.SALE, merchant_id="M12345", terminal_id="T67890"))
Swift
let eventId = try sdk.doTransaction(request: TransactionRequest(amount: 10000, type: .sale, merchantId: "M12345", terminalId: "T67890"))

lifecycle

init(config)

Initialize the Pine Labs SDK with configuration. Must be called exactly once per process before any other API.

Overview

Stabilitystable
Since0.1.0
ReturnsPinelabsSdk — the singleton SDK handle

Parameters

NameTypeRequiredDescription
configSdkConfig✅ YesSDK infrastructure configuration (transports, logging, cloud mode)

Example config: 
{ "transportPreference": ["TCP", "BLUETOOTH"], "logLevel": "INFO", "cloudResultMode": "MANUAL_POLL" }

Errors

VariantWhenRecoverable
AlreadyInitializedinit() called more than once in the same process❌ No
InvalidConfigAny config field fails validation❌ No

Code Examples

Kotlin
val sdk = PinelabsSdk.init(SdkConfig(transportPreference = listOf(TransportKind.TCP)))
Python
sdk = PinelabsSdk.init(SdkConfig(transport_preference=[TransportKind.TCP]))
Swift
let sdk = try PinelabsSdk.init(config: SdkConfig(transportPreference: [.tcp]))

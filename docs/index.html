🕐 Time, Date & Stock Price MCP Server

Get current IST time, IST date, and live company share prices — directly inside your AI assistant.

✅ MCP Registry Listed 🐳 Docker 🐍 Python ⚡ stdio transport

🛠️ Tools

🕐

get_current_time()

Returns the current time in IST (Indian Standard Time). e.g. 11:45:30 PM IST

📅

get_current_date()

Returns the current date in IST. e.g. Wednesday, 22 April 2026

📈

get_share_price(company_name)

Returns live stock price for any company by name. e.g. "Infosys", "Apple", "Pine Labs"

📋

list_pinelabs_apis()

Lists all available Pine Labs SDK APIs grouped by category. Call this first to discover API names.

📖

get_api_documentation(api_name)

Returns full Pine Labs SDK documentation for a specific API — parameters, errors, and code examples in Kotlin, Python, Swift.

🚀 Quick Install

Add to mcp.json at %APPDATA%\Code\User\mcp.json

{
  "servers": {
    "utility-mcp-server": {
      "type": "stdio",
      "command": "docker",
      "args": ["run", "--rm", "-i", "prakhargour/utility-mcp-server:1.0.1"]
    }
  }
}

Add to claude_desktop_config.json at %APPDATA%\Claude\

{
  "mcpServers": {
    "utility-mcp-server": {
      "command": "docker",
      "args": ["run", "--rm", "-i", "prakhargour/utility-mcp-server:1.0.1"]
    }
  }
}

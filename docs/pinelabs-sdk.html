init(config)

Initialize the Pine Labs SDK. Must be called once before any other API. Accepts transport preference, log level, and cloud result mode.

doTransaction(request)

Start a payment transaction on the terminal. Returns an eventId immediately. Final result delivered via callbacks.

checkStatus(eventId)

Poll the status of an in-flight or completed transaction by its eventId. Returns PENDING, IN_PROGRESS, SUCCESS, FAILED, or UNKNOWN.
